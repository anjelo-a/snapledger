# SnapLedger

Offline-first Android-native personal finance app with receipt scan, OCR, structured review, budgets, dashboard, and backend sync.

## Repo Layout

- `app/` Android app module (Kotlin + Compose)
- `backend/` FastAPI backend
- `docs/` planning and engineering documentation

## Open In Android Studio

Open this repository root directly in Android Studio.

## Development Setup

For full co-builder onboarding (Android <-> backend wiring, local/cloud DB, Cloud Run, envs, migrations, and troubleshooting), see:

- `docs/DEVELOPMENT_SETUP.md`
